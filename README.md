# KPSC-informatics-PEARL-BERT
Programs for BERT Model's developments in 'Cohort vs Case–Control Design for Transformer-Based Prediction of Asthma Exacerbations in Mild Asthma'
# Example of use
1. [Synthetic dataset](https://github.com/KPSCRE-Informatics/PEARL-BERT/blob/main/simulated_sample.csv) - generated by simulation for program test and demo purposes rather than performance accuracy. <br>
2. [BERT dataset format conversion sample code](https://github.com/KPSCRE-Informatics/PEARL-BERT/blob/main/pearl_bert_format_conversion.py) - convert dataset into bert word tokens format for model developments. <br>
3. [BERT model pretrain sample code](https://github.com/KPSCRE-Informatics/PEARL-BERT/blob/main/pearl_bert_pretrain.py) - pretrain bert model based on specific dataset/corpus for downstream classification. <br>
4. [BERT model fine-train sample code](https://github.com/KPSCRE-Informatics/PEARL-BERT/blob/main/pearl_bert_train.py) - fine train the bert model for asthma exacerbation classification/predication. <br>
5. [BERT modoel prediction sample code](https://github.com/KPSCRE-Informatics/PEARL-BERT/blob/main/pearl_bert_predict.py)- generate the predicted probability using the fine-tune bert model for given dataset. <br>
6. [BERT_model_performance_calculation sample_code](https://github.com/KPSCRE-Informatics/PEARL-BERT/blob/main/pearl_roc_auc.py) - calculate the AUC and draw ROC curves based on the generated probability to assess model performance.
